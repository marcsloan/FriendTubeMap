Putting your Facebook friends on the London Underground

A mashup created by Marc Sloan that takes a person's social network from Facebook and puts their friends as station names on the London Underground map. Check out the GitHub page.

Credit goes to a number of open source projects without which this project would not have been possible:

The tube map is a customized version of John Galantini's awesome CSS Tube Map

Grabbing social data would not be possible without the great NetGet resource. Sadly Facebook will be removing this functionality in April so this site won't be around for very long :(

The website was built using the excellent Tango With Django online resource

The graph package used was NetworkX and I used Thomas Aynaud's community package to find friend communities within the social network
